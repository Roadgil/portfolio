8개 모듈로
완성된 파이프라인

01
🖥️
import_manager_app_6in1_pretty.py
통합 런처 GUI
Tkinter로 만든 6-in-1 통합 실행 패널. 상태 패널에서 최신 파일 수정시각, 신규 추가 건수, 마지막 메일 발송 기록을 한눈에 확인하고 60초마다 자동 새로고침.
주요 경로 유효성 자동 체크
BAT 실행 후 5초 자동 상태 갱신
상태값 색상 구분 (초록/기본/빨강)
02
📧
outlook_intransit_downloader.py
Intransit 리포트 다운로더
Outlook 받은편지함 하위 폴더에서 최신 APAC Candela Intransit Details Report.xlsx를 자동으로 찾아 날짜 suffix를 붙여 저장.
최신 메일 1개만 선택 (중복 방지)
폴더 경로 계층 탐색 자동화
파일명 날짜 suffix 자동 부여
03
📄
outlook_imp_pdf_saver.py
수입신고필증 PDF 저장기
인천관세법인 메일 폴더에서 _IMP_ / IMP_ 프리픽스 PDF 첨부를 자동 저장. State JSON으로 중복 스캔 방지, 수정신고 시 덮어쓰기 정책 적용.
State JSON 기반 마지막 실행 시각 추적
최신→과거 정렬 + cutoff 이후만 스캔
동일 파일명 덮어쓰기 (수정신고 대응)
04
📊
intransit_to_ir_append_only_sorted_paths_set.py
Intransit → IR 신청목록 Append
가장 최신 Intransit xlsx를 자동 선택하여 [Intransit] 시트에 신규 데이터만 Ship Date 오름차순으로 append. 기존 행 순서는 절대 변경하지 않음.
중복 행 완전 차단 (Set 기반)
본사 선적일자 / 용마 입고 예정 자동 계산
실행 전 타임스탬프 백업 자동 생성
05
📑
pdf_auto_updater.py
PDF → 수입신고실적 업데이터
pdfplumber로 수입신고필증 PDF를 파싱하여 수입신고실적 Excel에 자동 반영. Hash 기반 처리 DB로 동일 PDF 중복 처리 완전 방지. 통관 알림 메일도 자동 발송.
SHA 해시 기반 _processed_pdfs.json DB
보호 컬럼 덮어쓰기 방지 (PROTECTED_COLS)
특정 자재코드 통관 알림 메일 자동 발송
06
🔄
auto_import_complete.py
수입신고실적 Append + Backfill
Intransit 데이터를 수입신고실적 Excel에 신규 행으로 추가하고, 기존 행의 빈 칸(SN/Lot, Serial Number)을 Backfill. Delivery/SO 매핑 시 유일 후보만 채워 오류를 방지.
Delivery 번호 숫자/문자 혼합 통일 인식
Backfill: 유일 후보만, 중복이면 스킵
동일 Intransit 내 중복 행 1회만 append
07
📮
intransit_internal_share.py
내부 공유 메일 발송기
Urgent Item 시트와 대조하여 요청 품목이 실렸는지 확인 후 서비스/영업팀 전체에 Intransit 현황 메일 자동 발송. 상태 파일 기반 중복 발송 완전 방지.
Urgent 매칭 없으면 Urgent 섹션 자동 숨김
동일 P/N 복수 요청 시 최신 Request 1건만
SharePoint 링크 포함 HTML 메일 자동 생성
08
🔁
intransit_update_runner_with_state.py
업데이트 러너 + 상태 추적
대상 스크립트를 subprocess로 실행하고 stdout을 파싱하여 실행 결과를 JSON으로 저장. CP949/UTF-8 인코딩 자동 감지로 한글 로그가 깨지지 않음.
CP949 우선 디코딩 + UTF-8 fallback
added_rows / max_ship / 파일경로 파싱
intransit_update_state.json 자동 갱신
